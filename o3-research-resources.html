💡Best Practices

Keep It Concise

o3 thrives with direct instructions without elaborate explanations

Avoid Chain-of-Thought

No need to say "think step-by-step" - it reasons internally

Use Delimiters

Use triple quotes, XML tags for organizing input parts

Limit Extra Context

Include only the most pertinent information to avoid overwhelming
